# Pasta de Arquivos Estáticos

## Como substituir a imagem do logo

1. **Coloque sua imagem** nesta pasta `static/`
2. **Renomeie** o arquivo para `imagem.jpg`
3. **Formatos suportados**: JPG, JPEG, PNG
4. **Tamanho recomendado**: 30x30 pixels (ou proporcional)

## Exemplo

- Coloque o arquivo `logo-da-associacao.jpg` na pasta `static/`
- Renomeie para `imagem.jpg`
- Reinicie a aplicação
